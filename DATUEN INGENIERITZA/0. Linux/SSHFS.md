
Windowsen debian makinako direktorio bat montatu nahi baduzu, komando honen bidez Z izeneko diskoa montatu dezakezu Windowsen VSCode-gaz editatu ahal izateko nano beharrean.
```
net use Z: \\sshfs.r\root@localhost!2209\root /user:root root
```

```
net use Z: \\sshfs.r\ibai@localhost!2205\home\ibai /user:ibai ibai
```
