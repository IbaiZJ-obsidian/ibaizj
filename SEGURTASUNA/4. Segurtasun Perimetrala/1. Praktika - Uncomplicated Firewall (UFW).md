
Lehenengo, zerbitzuak simulatu nc-rekin

```bash
# HTTP - 80
nc -lk -p 80

# HTTPS - 443
nc -lk -p 443

# MySQL - 3306
nc -lk -p 3306

# SSH simulatua - 2222 
nc -lk -p 2222

# OpenTTD - 3979 TCP
nc -lk -p 3979
# OpenTTD - 3979 UDP
nc -lku -p 3979
```

UFW erregelak konfiguratu

```bash
# UFW aktibatu
sudo ufw enable

# Politika lehenetsia: dena blokeatu
sudo ufw default deny incoming
sudo ufw default allow outgoing

# SSH (2222): IP bakar batetik soilik
sudo ufw allow from 192.168.0.10 to any port 2222

# HTTP eta HTTPS: IP guztietatik
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp

# MySQL: erregistratu baina ez blokeatu (LOG)
sudo ufw allow log 3306/tcp

# OpenTTD: sare lokaletik bakarrik (VMko sarea)
sudo ufw allow from 192.168.0.0/24 to any port 3979

# Gainerako guztia blokeatu (jada default deny-k egiten du)
```

Probak egin

```bash
# Nmap-ekin portuen egoera ikusi
nmap -sT localhost
nmap -sU localhost  # UDP portuak

# Netcat bezero gisa
nc -v 127.0.0.1 80   # konektatu beharko da
nc -v 127.0.0.1 3306 # konektatu eta LOG sartu beharko da
nc -v 127.0.0.1 2222 # ezin konektatu IP okerra bada
```
